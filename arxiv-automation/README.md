# arXiv CV Papers - Video Generation & Editing

每日自动更新 arXiv 计算机视觉相关论文，专注于视频生成（Video Generation）和视频编辑（Video Editing）领域。

## 📋 功能特性

- **自动抓取**：每天北京时间 10:00 自动运行
- **多领域覆盖**：cs.CV, cs.AI, cs.MM, cs.RO, cs.LG
- **智能筛选**：自动筛选 video generation 和 video edit 相关论文
- **去重机制**：与最近 5 天内容对比，避免重复
- **双语支持**：提供中英文摘要
- **资源链接**：包含论文 PDF、项目主页、代码仓库链接
- **双版本模式**：
  - **V1**：基础版，提供论文标题、摘要、链接和翻译
  - **V2**：增强版，包含 AI 全文分析和批判性评价

## 📖 使用方法

### 切换版本

在仓库的 **Settings → Secrets and variables → Actions → Variables** 中设置：

- 变量名：`VERSION`
- 变量值：`v1` 或 `v2`

### 手动触发

进入 **Actions** 标签页，选择 `Daily arXiv Papers Update` 工作流，点击 `Run workflow` 手动触发。
